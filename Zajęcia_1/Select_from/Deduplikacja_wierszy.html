Deduplikacja wierszy

Created sobota 04 czerwca 2022

Aby usunąć z zapytania duplikaty wierszy możemy użyć SELECT DISTINCT lub UNION 

SELECT DISTINCT * 
FROM table;

lub 

(SELECT * FROM table LIMIT 1)
UNION
SELECT * FROM table

Nie należy używać UNION ALL w tym celu, gdyż to wyłączy usuwanie duplikatów.
